# Function that computes various variant statistics from a data frame with variant records

Function that computes various variant statistics from a data frame with
variant records

## Usage

``` r
get_variant_statistics(var_df = NULL, pct_other_limit = 4)
```

## Arguments

- var_df:

  data frame with variants

- pct_other_limit:

  numeric value specifying the percentage limit for the 'Other' category
