# Function that assigns a SOMATIC_CLASSIFICATION to variants based on evidence found in variant set, potentially limited by user-defined options

Function that assigns a SOMATIC_CLASSIFICATION to variants based on
evidence found in variant set, potentially limited by user-defined
options

## Usage

``` r
assign_somatic_classification(sample_calls, settings)
```

## Arguments

- sample_calls:

  data frame with putative somatic variants

- settings:

  PCGR run/configuration settings

## Value

sample_calls
