# Function that checks for recurrence of RNA fusions in Mitelman database and annotates with clinical and cytogenetic features

Function that checks for recurrence of RNA fusions in Mitelman database
and annotates with clinical and cytogenetic features

## Usage

``` r
rna_fusion_recurrence_mitdb(query_fusions = NULL, ref_data = NULL)
```

## Arguments

- query_fusions:

  data frame with RNA fusion calls

- ref_data:

  PCGR reference data bundle (list)

## Value

data frame with RNA fusion calls annotated with Mitelman database
features
