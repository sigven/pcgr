# Function that assigns a logical (STATUS_DBSNP) reflecting whether a variant co-incides with an entry in dbSNP (germline)

Function that assigns a logical (STATUS_DBSNP) reflecting whether a
variant co-incides with an entry in dbSNP (germline)

## Usage

``` r
dbsnp_germline_status(sample_calls)
```

## Arguments

- sample_calls:

  data frame with sample variant calls
