# Function that retrieves clinical evidence items (CIVIC, CBMDB) for CNA aberrations

Function that retrieves clinical evidence items (CIVIC, CBMDB) for CNA
aberrations

## Usage

``` r
get_clin_assocs_cna(onco_ts_sets, annotation_tags = NULL, eitems = NULL)
```

## Arguments

- onco_ts_sets:

  data frame with annotations with respect to lost tumor suppressor
  genes and gained oncogenes

- annotation_tags:

  list with annotation tags for display in report

- eitems:

  Data frame with clinical evidence items

## Value

list
