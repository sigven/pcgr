# Function that initiates PCGR/CPSR report object

Function that initiates PCGR/CPSR report object

## Usage

``` r
init_report(yaml_fname = NULL, report_mode = "PCGR")
```

## Arguments

- yaml_fname:

  Yaml file with configuration settings and paths to annotated molecular
  datasets

- report_mode:

  Type of report ('PCGR' or 'CPSR')
