# Function that structures variant evidence items according to strength of evidence

Function that structures variant evidence items according to strength of
evidence

## Usage

``` r
structure_var_eitems(var_eitems, annotation_tags, alteration_type = "MUT")
```

## Arguments

- var_eitems:

  variant evidence items

- annotation_tags:

  annotation tags to include for display

- alteration_type:

  type of alteration ('MUT','CNA')
