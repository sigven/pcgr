# Function that assigns a logical (STATUS_LIKELY_GERMLINE_HOMOZYGOUS) reflecting whether a variant is likely homozygous (germline) - based on allelic fraction (VAF_TUMOR)

Function that assigns a logical (STATUS_LIKELY_GERMLINE_HOMOZYGOUS)
reflecting whether a variant is likely homozygous (germline) - based on
allelic fraction (VAF_TUMOR)

## Usage

``` r
hom_af_status(sample_calls)
```

## Arguments

- sample_calls:

  data frame with sample variant calls
