# Function that loads YAML data with settings and file paths to annotated molecular profiles

Function that loads YAML data with settings and file paths to annotated
molecular profiles

## Usage

``` r
load_yaml(yaml_fname, report_mode = "CPSR")
```

## Arguments

- yaml_fname:

  Yaml file with configuration settings and paths to annotated molecular
  datasets

- report_mode:

  Type of report ('PCGR' or 'CPSR')
