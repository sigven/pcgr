# Function that assigns a logical (STATUS_COSMIC) reflecting whether a variant co-incides with an entry in COSMIC (germline)

Function that assigns a logical (STATUS_COSMIC) reflecting whether a
variant co-incides with an entry in COSMIC (germline)

## Usage

``` r
cosmic_somatic_status(sample_calls)
```

## Arguments

- sample_calls:

  data frame with sample variant calls
