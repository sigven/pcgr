# Get BSgenome Object

Gets BSgenome object given a human genome assembly string.

## Usage

``` r
get_genome_obj(genome)
```

## Arguments

- genome:

  Human genome assembly string: hg38 or hg19.

## Value

BSgenome object.

## Examples

``` r
if (FALSE) { # \dontrun{
get_genome_obj("hg38")
} # }
```
