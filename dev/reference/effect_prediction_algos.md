# List of URLs for a range of variant effect prediction algorithms

List of URLs for a range of variant effect prediction algorithms

## Usage

``` r
effect_prediction_algos
```

## Format

A data.frame with 21 rows and 3 columns that indicates URL's for variant
effect prediction algorithms

- *algorithm* - Name encoding for effect prediction algorithm

- *url* - URL

- *display_name* - Display name for use in reporting
