# Render actionability tier as a styled badge with tier-specific colors

(Generated by Claude Opus 4.6 with some manual tweaks)

## Usage

``` r
render_tier_cell(tier_letter = "T", tier_colors = NULL)
```

## Arguments

- tier_letter:

  Letter prefix for the tier (e.g. "T" for sensitivity, "R" for
  resistance)

- tier_colors:

  Named list with `t1` and `t2` hex colors for tiers 1 and 2

A function that can be used as a reactable cell renderer for the
