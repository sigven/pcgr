# Function that retrieves relevant (interventional based on molecular target) clinical trials for a given tumor type

Function that retrieves relevant (interventional based on molecular
target) clinical trials for a given tumor type

## Usage

``` r
generate_report_data_trials(pcgr_data, config, sample_name)
```

## Arguments

- pcgr_data:

  PCGR data bundle object

- config:

  PCGR run configurations

- sample_name:

  sample name

## Value

pcg_report_trials data frame with all report elements
