# Function that assigns a logical to STATUS_CLINVAR_GERMLINE based on whether a ClinVar entry of germline origin is found for a given variant (for entries in a data frame)

Function that assigns a logical to STATUS_CLINVAR_GERMLINE based on
whether a ClinVar entry of germline origin is found for a given variant
(for entries in a data frame)

## Usage

``` r
clinvar_germline_status(sample_calls)
```

## Arguments

- sample_calls:

  data frame with sample calls
