# Fixed data types and levels for biomarker evidence items

Fixed data types and levels for biomarker evidence items

## Usage

``` r
biomarker_evidence
```

## Format

A named list with three character vectors:

- *types* - Biomarker evidence types (e.g. "predictive", "prognostic")

- *levels* - Evidence strength levels (e.g. "A: Validated", "B: Clinical
  evidence")

- *clinical_significance* - Clinical significance categories (e.g.
  "Sensitivity/Response")
