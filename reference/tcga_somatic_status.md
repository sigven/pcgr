# Function that assigns a logical (STATUS_TCGA_SOMATIC) reflecting whether a variant co-incides with an entry in TCGA (somatic)

Function that assigns a logical (STATUS_TCGA_SOMATIC) reflecting whether
a variant co-incides with an entry in TCGA (somatic)

## Usage

``` r
tcga_somatic_status(sample_calls)
```

## Arguments

- sample_calls:

  data frame with sample variant calls
