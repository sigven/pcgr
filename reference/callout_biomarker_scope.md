# Emit the shared "biomarker types and report scope" callout note

Emit the shared "biomarker types and report scope" callout note

## Usage

``` r
callout_biomarker_scope(variant_label = "variants")
```

## Arguments

- variant_label:

  Variant-type phrase for the Tier III sentence, e.g. "SNV/InDel
  variants", "copy number aberrations", "RNA fusions".
