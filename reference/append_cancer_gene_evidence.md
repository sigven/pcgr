# Function that appends cancer gene evidence links

Function that appends cancer gene evidence links

## Usage

``` r
append_cancer_gene_evidence(var_data_df = NULL, ref_data = NULL)
```

## Arguments

- var_data_df:

  Data frame of sample variants from VCF

- ref_data:

  PCGR/CPSR reference data object

## Value

var_data_df
