# Character vector of tumor site names used in PCGR/CPSR reports

Character vector of tumor site names used in PCGR/CPSR reports

## Usage

``` r
tumor_sites
```

## Format

A character vector of 31 tumor site names (e.g. "Any", "Adrenal Gland",
"Biliary Tract"). Used to constrain tumor site selection in
configuration.
