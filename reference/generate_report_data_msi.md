# Function that generates MSI prediction data for PCGR report

Function that generates MSI prediction data for PCGR report

## Usage

``` r
generate_report_data_msi(variant_set, ref_data = NULL, settings = NULL)
```

## Arguments

- variant_set:

  variant calls subject to MSI classification

- ref_data:

  PCGR reference data object

- settings:

  PCGR run/configuration settings
