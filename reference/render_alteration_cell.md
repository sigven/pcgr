# Render sample alteration with confidence icon Uses filled circle + bold for high confidence, hollow circle + muted for medium confidence

(Generated by Claude Opus 4.6 with some manual tweaks)

## Usage

``` r
render_alteration_cell(data)
```

## Arguments

- data:

  The main data frame (passed via closure)

## Value

A function that can be used as a reactable cell renderer for the
SAMPLE_ALTERATION column, which checks the BM_TOP_MAPPING_CONFIDENCE
column in the same row to determine styling.

High confidence: filled circle (e.g. \#37303A) + bold text Medium
confidence: hollow circle (border \#999999) + normal weight + muted text
color
