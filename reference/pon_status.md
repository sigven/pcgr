# Function that assigns a logical (STATUS_PON) reflecting whether a variant is co-inciding with a variant present in a panel-of-normals database (PANEL_OF_NORMALS column is TRUE)

Function that assigns a logical (STATUS_PON) reflecting whether a
variant is co-inciding with a variant present in a panel-of-normals
database (PANEL_OF_NORMALS column is TRUE)

## Usage

``` r
pon_status(sample_calls)
```

## Arguments

- sample_calls:

  data frame with sample variant calls
