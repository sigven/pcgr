# Function that adds TCGA annotations (cohort, frequency etc.) to variant identifiers

Function that adds TCGA annotations (cohort, frequency etc.) to variant
identifiers

## Usage

``` r
append_tcga_var_link(var_df, linktype = "dbsource")
```

## Arguments

- var_df:

  data frame with variants

- linktype:

  type of link

## Value

var_df
