# Named list of biomarker categories used for variant classification

Named list of biomarker categories used for variant classification

## Usage

``` r
bm_categories
```

## Format

A named list with six elements, each representing a biomarker category
(e.g. "therapeutic_sensitivity", "prognostic_poor"). Each element
contains:

- *etype* - Evidence type (e.g. "predictive", "prognostic",
  "diagnostic")

- *clnsig* - Clinical significance label(s) for the category
