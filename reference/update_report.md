# Function that updates a PCGR/CPSR report object structure

Function that updates a PCGR/CPSR report object structure

## Usage

``` r
update_report(report, report_data, a_elem = "snv_indel")
```

## Arguments

- report:

  PCGR/CPSR report

- report_data:

  Object with report contents

- a_elem:

  section of PCGR report
