# Function that styles CNA variant classes (amplification, gain, hetdel, homdel/hemdel) with more informative names for display in PCGR reports and other outputs

Function that styles CNA variant classes (amplification, gain, hetdel,
homdel/hemdel) with more informative names for display in PCGR reports
and other outputs

## Usage

``` r
append_styled_cna_vclass(var_data_df = NULL, colname = "VARIANT_CLASS_STYLED")
```

## Arguments

- var_data_df:

  Data frame of CNA calls

- colname:

  Name of column to be styled and appended to var_data_df

## Value

var_data_df
