# Function that parses and loads reference data from files in the assembly-specific PCGR bundle directory

Function that parses and loads reference data from files in the
assembly-specific PCGR bundle directory

## Usage

``` r
load_reference_data(pcgr_db_assembly_dir = NULL, genome_assembly = "grch38")
```

## Arguments

- pcgr_db_assembly_dir:

  Assembly-specific root path for data bundle

- genome_assembly:

  grch37/grch38
