# Function that adds GWAS citation/phenotype to GWAS hit found through PCGR annotation

Function that adds GWAS citation/phenotype to GWAS hit found through
PCGR annotation

## Usage

``` r
append_gwas_citation_phenotype(vcf_data_df = NULL, ref_data = NULL)
```

## Arguments

- vcf_data_df:

  Data frame of sample variants from VCF

- ref_data:

  PCGR/CPSR reference data object

## Value

vcf_data_df
