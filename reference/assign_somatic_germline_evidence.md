# Function that appends several tags denoting evidence for somatic/germline status of variants

Function that appends several tags denoting evidence for
somatic/germline status of variants

## Usage

``` r
assign_somatic_germline_evidence(sample_calls, settings = NULL)
```

## Arguments

- sample_calls:

  data frame with variants

- settings:

  PCGR run/configuration settings

## Value

sample_calls
