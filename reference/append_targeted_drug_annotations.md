# Function that adds link to targeted drugs (on and off-label) for a list of variants and associated targeted

Function that adds link to targeted drugs (on and off-label) for a list
of variants and associated targeted

## Usage

``` r
append_targeted_drug_annotations(
  var_df,
  primary_site = "Breast",
  ref_data = NULL
)
```

## Arguments

- var_df:

  data frame with variants

- primary_site:

  primary site of tumor sample

- ref_data:

  PCGR/CPSR reference data object
